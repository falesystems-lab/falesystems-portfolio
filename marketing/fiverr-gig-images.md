## Design Guidelines for All Images

### Typography:
- **Titles:** 60-80px, bold, all caps
- **Subtitles:** 28-36px, regular weight
- **Body text:** 20-24px
- Always use high contrast (white on dark)

### Layout:
- **Rule of thirds:** Keep important text/icons in top-left or center
- **Padding:** 80-100px from edges (text won't get cut off in thumbnails)
- **Focal point:** One main element per image (don't overcrowd)

### Colors:
- **Primary:** White text on dark background
- **Accents:** Use brand orange/red sparingly (highlights, badges, borders)
- **Never:** All-white or all-black images (low contrast)

### Icons/Graphics:
- Use emojis (ðŸ”¥ ðŸ¤– âœï¸ ðŸ•·ï¸) - instantly recognizable
- Or simple line icons (Heroicons, Lucide)
- Consistent style across all images

### Text Overlays:
- Always use background rectangles/boxes for text (readability)
- Semi-transparent dark backgrounds work well
- Drop shadows on text (subtle, 2-3px)

## Tools to Create Images (Free Options)

1. **Canva** (easiest, templates available)
   - Use 1920x1080 custom size
   - Free templates for "Fiverr gig images"
   - Export as PNG

2. **Figma** (more control)
   - Free for individual use
   - Professional design tool
   - Easy collaboration

3. **Photopea** (Photoshop alternative, browser-based)
   - Free
   - Works in browser
   - Supports layers, gradients, etc.

4. **HTML/CSS** (if you're technical)
   - Create using code (like we did with logo)
   - Screenshot at 1920x1080
   - Full control over design

## Image Creation Workflow

1. **Create template** (Canva/Figma)
   - Set up 1920x1080 canvas
   - Add brand colors, logo, fonts
   - Save as reusable template

2. **Duplicate for each gig**
   - Change text, icons, colors per gig
   - Maintain consistent style

3. **Export & optimize**
   - Export as PNG (high quality)
   - Compress (TinyPNG.com)
   - Check file size < 5MB

4. **Test thumbnail**
   - View at small size (200x113px)
   - Text still readable?
   - Logo/icon still visible?

## A/B Testing Strategy

**After initial upload:**
- Track which gigs get more clicks
- Test variations:
  - Different headline phrasing
  - Different colors (orange vs blue vs green)
  - Price visible vs hidden
  - Emoji vs no emoji

**Fiverr shows analytics:**
- Click-through rate
- Impressions
- Orders per view

Iterate based on data.
